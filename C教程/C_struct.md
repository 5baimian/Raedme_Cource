# 一、结构体定义与使用
## 区分结构体类型与结构体变量
####  1.结构体类型可以进行重命名，而结构体变量不能重命名
```c++
struct Student { // 结构体类型
    int age;
    char name[20];
};
struct Student stu1; // 结构体变量
```
#### 2.在调用结构体时记得包含其所在的头文件
#### 3.结构体的嵌套，并且在结构体中可以创建枚举变量
